mantou
=============

##使用：
1. 使用前准备：npm install && bower install
2. 起数据库：mongod -dbpath d:/mongodb/mantou
3. 起服务：gulp

##基于：
* nodejs
* mongodb
* gulp
* express
* mongoose
* Handlebars
* sass
* requirejs
* jQuery
* bootstrap
